# ZigZag + Revert Fibo Expansion

Индикатор для MetaTrader 4, объединяющий ZigZag с расширениями Фибоначчи.

## Основные функции
- Построение ZigZag с настраиваемыми параметрами
- Автоматическое построение расширений Фибоначчи
- Визуальные сигналы (стрелки и линии)
- Три типа уровней Фибоначчи

## Параметры настройки

### ZigZag
- BR = 500 (количество баров)
- LG = 6 (баров для M5+)
- LG2 = 15 (баров для M1)
- LT = 5 (толщина линии)

### Визуальные
- SCLR = Magenta (цвет продажи)
- BCLR = Green (цвет покупки)
- LCLR = Blue (цвет ZigZag)
- ARS = 5 (размер стрелок)
- LARS = 5 (толщина линий)

### Фибоначчи
- FiboExp = true (включить Фибо)
- FiboLevelsType = FIBO_SHORT (тип уровней)
- ShowArrow = true (показывать стрелки)
- ShowHLine = true (показывать линии)

## Установка
1. Скопируйте файл в папку MQL4/Indicators
2. Перезапустите MT4
3. Назначьте индикатор на график

## Особенности
- Автоподстройка под таймфрейм M1
- Три набора уровней Фибоначчи
- Оптимизированная производительность

- Индикатор трендовый.
Если переворот был на уровне от 0 до 38.2, то Тейк Профит выставляем на уровне 76.4;
